== Description ==

Dummy payment gateway for developers. Simulates IPN. Always marks Order as paid.

Useful for testing and developing. Never use it on live sites!

== Installation ==

As any other plugin, via WordPress admin panel, or upload manually to the `/wp-content/plugins/` directory.


== Frequently Asked Questions ==

= Is this plugin useful for real / public / producion sites? =

No.

= Does this plugin do any wierd workflow with Order / WooCommerce? =

No, it uses standard Paymnet Gateway API provided by WooCommerce.
